# GitHub Assignment

## What is Git?
Git is a version control system used to track changes in files.
It helps developers manage different versions of a project safely.

## What is GitHub?
GitHub is an online platform that hosts Git repositories.
It allows users to store code remotely and share projects.

This repository was created to demonstrate my ability to:
- Create a GitHub repository
- Add files
- Commit changes
- Push changes to a remote repository
